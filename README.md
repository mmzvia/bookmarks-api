# Nest.js REST API

## Description

A simple RESTful API that allows users to register accounts and manage their personal bookmarks.

## Scripts

```bash
# Project setup.
$ npm install

# Run e2e tests.
$ npm run test:e2e

# Run the development server.
$ npm run start
```

## Scenarios

**Registration & Login**

1. Register a new user account.
2. Log in to an existing user account.

**Create & Manage Bookmark**

1. Create a new bookmark.
2. Retrieve all bookmarks.
3. Retrieve a specific bookmark by its ID.
4. Update an existing bookmark.
5. Delete a specific bookmark.

## Project Modules

### Authentication

- User registration.
- User login.

### Users

- Get account info.
- Update account info.
- Delete account.

### Bookmarks

- Create a new bookmark.
- Get all user's bookmarks.
- Get a specific bookmark by ID.
- Update a bookmark.
- Delete a bookmark.
